# 说明

本项目是使用vue + elementUI  搭建的一个纯前端的小型汽车管理系统。初始数据采用easy-mock模拟，但是发现easy-mock的官网经常打不开，可能是服务器原因，因此我数据用的是我自己本地搭建的easy-mock数据，有兴趣的朋友也可以自己本地搭建easy-mock，方便使用。
